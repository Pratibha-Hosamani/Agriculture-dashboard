# Project Description
  This project analyzes agriculture production data using AWS,Snowflake and power BI.The data is stored in AWS S3 and processed in snowflake. An interactive power    BI dashoard is created to visualize crop production, state-wise performance and yearly trends.
## Objectives
  - Analyze crop production performance
  - Compare production across different states and regions
  - Identify high performing crops
  - Understand agricultural trends over time
  - Create a dashboard for easy data visualization
## Tools used
 - AWS S3
 - Snowflake
 - Power BI
 - SQL
## Dataset Information
  The dataset contains information about:
  - Year
  - Location
  - Area
  - Rainfall
  - Temperature
  - Soiltype
  - Irrigation
  - yeilds
  - Humidity
  - Crops
## Data Analysis Process
  - Uploaded the dataset to AWS S3 for storage.
  - Connected Snowflake to AWS S3 and loaded the data into Snowflake tables.
  - Used SQL queries in Snowflake to clean and analyze the data.
  - Connected Power BI to Snowflake for visualization.
  - Created an interactive dashboard to show agricultural insights.
## Key Insights
   - Some crops show higher production compared to others.
   - Certain states contribute more to agricultural production.
   - Production trends vary across different years.
   - Yield performance helps identify efficient crop production.
## Dashboard Features
   - Total Production Overview
   - Production by Crop Type
   - Production by State
   - Yearly Production Trends
   - Yield Analysis Dashboard
## Conclusion
   This project helps to understand agriculture performance and crop production trends. The dashboard provides insights that can support better agricultural 
   planning and decision making.
