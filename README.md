# Project Description
This project analyzes agriculture production data using AWS,Snowflake and power BI.The data is stored in AWS S3 and processed in snowflake. An interactive power BI dashoard is created to visualize crop production, state-wise performance and yearly trends.
## Tools used
 - AWS S3
 - Snowflake
 - Power BI
### Key Features 
 - Analysis of crop production data
 - State-wise agriculture performance
 - Yearly production trends
 - Interactive Power BI dashboard for better insights
#### Outcome
The dashboard helps to easily understand agriculture perfotmance and identify trends in crop production using data visualisation
